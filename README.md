🚀 SEO & Citation Agent 

An intelligent, automated agent designed to analyze, optimize, and manage SEO workflows.
This project leverages LLMs and specialized tools to provide actionable insights, keyword analysis, and content improvements.

⚡ Quick Start

Get the project running in minutes 👇

📥 Clone the repository
git clone https://github.com/Nandini56-bn/Hackathon-project.git
cd seo_agent-main
🛠 Run Setup
Windows: Double-click setup.bat
OR manual install:
pip install -r requirements.txt
🔐 Configure Environment
Rename .env.example → .env
Add your API keys (LLM providers, etc.) 🔑
▶️ Launch Application
run_all.bat

OR

python run_all.py

👉 This starts both backend + frontend together 🚀
