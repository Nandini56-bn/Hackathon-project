🔌 API ENDPOINTS QUICK REFERENCE
════════════════════════════════════════════════════════════════════════════════

Analyze Website
POST /api/analyze
Input:  { "website_url": "https://...", "simulate_improvement": true }
Output: { "seo_score": 65, "issues": [...], "recommendations": [...] }

Memory Status
GET /api/memory/example.com
Output: { "analyses_performed": 3, "improvement_percentage": 37.7 }

Simulate Improvements
POST /api/simulate-improvement
Input:  { "website_url": "https://...", "days": 5 }
Output: { "keywords_improved": {"seo": {...}} }

Batch Analyze
POST /api/batch-analyze
Input:  { "websites": ["url1", "url2", ...] }
Output: { "results": [...] }

View All Endpoints
GET /api/docs
Browser: Auto Swagger documentation
